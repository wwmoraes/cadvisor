## üßê About

Multi-platform cadvisor container built with support for:

- amd64 (x86_64/x64)
- arm64 (aarch64/armv8)
- armv7 (usually armhf, 32-bit ARMs with Thumb-2 and VFPv3)
- armv6 (usually armel, 32-bit ARMs with Thumb and VFPv2)
